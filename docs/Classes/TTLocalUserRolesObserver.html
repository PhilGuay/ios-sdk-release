TTLocalUserRolesObserver Class Reference

Inherits from NSObject
Declared in TTLocalUserRolesObserver.h

Overview

Provides a simple API for retrieving roles owned by the local user and watching for changes to those roles. This class should only be created, used, and deallocated on the main thread!
